## [1.0.0] - 2025-01-08

### Added
- Initial release of StockAnalytica
- User authentication (register/login) with JWT
- Stock browsing with sector filtering
- Buy stock functionality with 4% commission
- Portfolio management with real-time P&L tracking
- Transaction history
- MongoDB database integration
- RESTful API backend
- React frontend with Tailwind CSS
- Dark theme UI
- Risk indicators (High/Medium/Low)
- Search functionality for stocks
- Profile types (Focused/Diversified)

### Security
- Password hashing with bcrypt
- JWT token-based authentication
- CORS protection
- Environment variable configuration

## [0.5.0] - 2024-12-20

### Added
- Frontend UI design
- Authentication screens
- Stock card components
- Portfolio dashboard layout

### Changed
- Improved responsive design
- Enhanced color scheme

## [0.1.0] - 2024-12-01

### Added
- Project initialization
- Basic folder structure
